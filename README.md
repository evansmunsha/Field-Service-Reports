# Field Service Report

A free, open-source web app for recording and reviewing monthly field service activity.

Field Service Report helps users log time, track Bible studies, add notes, and generate clear monthly reports. It is designed to be simple, respectful, and private, with no payments, subscriptions, or locked features.

---

## Purpose

Many people need a clean and reliable way to keep accurate field service records without spreadsheets, paperwork, or paid apps. This project provides a modern, easy-to-use solution that anyone can use or self-host.

---

## Features

- Log daily service time with start and end times
- Track Bible studies and participants
- Add optional comments for each entry
- View monthly totals and summaries
- Generate printable and downloadable reports
- Mobile-friendly and desktop-friendly UI
- Secure authentication
- Fully open source

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Authentication:** Stack Auth
- **Database:** PostgreSQL (Neon) with Prisma
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **Date Utilities:** date-fns
